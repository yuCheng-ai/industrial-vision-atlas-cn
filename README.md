# 工业视觉选型地图 · Industrial Vision Atlas CN

面向自动化工程师、设备厂、视觉工程师的工业视觉选型知识库。

目标：不是做产品目录，而是把工业视觉方案设计过程结构化。

## 核心链路

```
检测需求
  ↓
工件特征
  ↓
成像方式
  ↓
光源
  ↓
镜头
  ↓
相机
  ↓
算法
  ↓
部署方案
```

## 内容规划

- 检测场景选型
- 相机选型
- 镜头选型
- 光源选型
- 3D视觉选型
- 算法路线
- 工业接口
- 行业方案
- 常见失败原因
- 视觉术语表
